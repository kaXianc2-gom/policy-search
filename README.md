# 🔍 PolicySearch — 政策文件搜索

[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)
[![Version](https://img.shields.io/badge/version-v1.0.0-green)](https://github.com/kaXianc2-gom/policy-search/releases)

> 内置 8 份核心政策文档 + 联网抓取 + 粘贴即搜。逐字模糊匹配 + 20 组同义词扩展 + 关键词高亮。

## 🎯 解决的痛点

同一政策省市县三级各有版本，精确关键词经常搜不到。PolicySearch 用逐字跳跃模糊匹配（"政法" → "政治学类、法学类"）+ 同义词扩展（"乡村振兴" → "农业农村现代化"），帮你找到相关内容。

**明确说明：这是关键词 + 同义词搜索，不是 AI 语义搜索。**

## ✨ 功能

### 🔍 智能搜索
- **逐字跳跃模糊匹配**：与 SheetLens 同款算法，字符级"政法"匹配"政治学类、法学类"
- **同义词扩展**：20 组同义词词典，自动扩展搜索词
- **搜索结果高亮**：匹配关键词黄色标记
- **相关度排序**：标题命中加权

### 📚 内置文档（8 份）
| 文档 | 年份 | 分类 |
|------|------|------|
| 党的二十大报告 | 2022 | 综合 |
| 2024 年政府工作报告 | 2024 | 经济 |
| 2024 年中央一号文件 | 2024 | 三农 |
| 公务员法（摘录） | 2018 | 法治 |
| 中国式现代化重要论述 | 2023 | 综合 |
| 数字中国建设整体布局规划 | 2023 | 科技 |
| 关于全面推进美丽中国建设的意见 | 2024 | 环保 |
| 党和国家机构改革方案（摘录） | 2023 | 政治 |

### 🌐 联网抓取
- 预设 6 个政府网站 URL（gov.cn）
- 自动抓取 + HTML 解析 + 加入搜索索引
- 双 CORS 代理自动切换

### 📋 粘贴即搜
- 从任何网站复制政策文本
- 粘贴即可加入搜索索引
- 适用场景：PDF 转换后的文本、公众号文章、内部文件

### 📂 分类浏览
- 按 7 大分类筛选：综合 / 经济 / 三农 / 法治 / 科技 / 环保 / 政治
- 点击文档卡片查看全文

## 🚀 快速开始

1. [下载 index.html](https://github.com/kaXianc2-gom/policy-search/releases/latest)
2. 双击打开
3. 输入关键词（如"乡村振兴 基层治理"）→ 回车
4. 点「📋 粘贴」粘贴外部文本 → 加入搜索

## 🔧 技术架构

| 项目 | 说明 |
|------|------|
| 搜索引擎 | 逐字跳跃模糊匹配（fuzzyMatch） |
| 同义词 | 20 组 / 80+ 条映射 |
| 联网 | fetch + CORS 代理 + HTML parse |
| 依赖 | 零外部依赖，纯 vanilla JS |

## ⚠ 已知限制

- 联网抓取依赖 CORS 代理，政府网站可能有反爬措施
- 此非 AI 语义搜索，不支持"大白话问问题"
- 建议使用「📋 粘贴」功能作为联网的可靠替代
